### Installation and Setup
Clone the project to your local machine
```bash
git clone https://github.com/shindesharad71/Anstagram.git
```

Change the directory to ```Anstagram```
```bash
cd Anstagram
```

Install the packages required
```bash
npm install
```

Start the application
```bash
npx nx serve
```
